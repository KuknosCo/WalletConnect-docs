# Set Network

You can switch between the public network and the test network by this method.<br />


### Example


<div >

``` javascript
    wallet.setNetwork(network.test) 
```
</div>

#### Params 

<table >
  <tr>
    <th>Name</th>
    <th>Type</th>
    <th>Description</th>
  </tr>
  <tr>
    <td>net</td>
    <td>walletConnect.network</td>
    <td>Network that you want work with it </td>
  </tr>
</table>


#### Return 

<div >
  void
</div>
